
# Unity new input system (InputAction)

## add inputAction to project
- Window > PackageManager
- Select "Unity Registry" on Dropdown beside Plus sign on most top-left
- type "Input System" in search bar
- install Input System
- after install unity required to restart the editor

## Create InputActionAsset
- right click on blank space in Project tab > Create > InputAction
- after named the asset ,double click on it
